# ℹ️ Information Website

A multi-page informational website created as part of **T INFO 230 – Web Development** coursework at the **University of Washington Tacoma**.

This assignment focuses on organizing content into structured webpages and connecting them through navigation while applying foundational web development concepts.

## 📖 Project Overview

The Information Website demonstrates how structured webpages can present educational or informational content in an organized and accessible way.

The project focuses on building a **multi-page static website** that includes:

- a homepage introducing the topic
- multiple informational pages
- navigation between pages
- organized media and image assets
- consistent layout and page structure

The goal of the assignment is to practice **structuring information for the web using HTML and CSS**.
