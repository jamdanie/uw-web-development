# Hello World – HTML Assignment

This project is an introductory exercise created for **T INFO 230 – Web Development** at the **University of Washington Tacoma**.

The assignment demonstrates the fundamental structure of an HTML document and basic webpage elements.

---

## 🌐 Live Demo

https://jamdanie.github.io/uw-web-development/assignments/hello-world/

---

## 📄 Project Overview

The goal of this assignment was to practice building a basic webpage using standard HTML elements.

The page demonstrates:

- HTML document structure
- headings and paragraph elements
- basic text formatting
- simple page layout concepts

This project serves as the starting point for learning how webpages are constructed and displayed in a browser.
