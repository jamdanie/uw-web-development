# 📚 Book Club Website

This project is a small themed website created as part of **T INFO 230 – Web Development** coursework at the **University of Washington Tacoma**.

The site demonstrates foundational front-end development skills using structured HTML, images, and basic page layout techniques.

---

## 🌐 Live Demo

View the project here:

https://jamdanie.github.io/uw-web-development/assignments/book-club/

---

## 📖 Project Overview

The Book Club website introduces a fictional book club and showcases:

- a homepage introducing the club
- visual elements such as book cover imagery
- structured page layout using semantic HTML
- simple branding elements such as logos and headers

The project focuses primarily on **content structure and basic webpage composition**.

---

## ⚙️ Technologies Used

This project was built using:

- **HTML5**
- basic page layout techniques
- static image assets
- simple content organization

Development tools included:

- **Visual Studio Code / Notepad++**
- **Git**
- **GitHub**
- modern web browser developer tools
