# 💪 Fitness Website

This project is a multi-page fitness themed website created as part of **T INFO 230 – Web Development** coursework at the **University of Washington Tacoma**.

The site demonstrates building a structured multi-page website with organized assets, external stylesheets, and navigation between pages.

---

## 🌐 Live Demo

View the project here:

---

## 🏋️ Project Overview

The Fitness Website simulates a small fitness or gym information site that includes several pages describing different aspects of a fitness program.

The website includes pages such as:

- Home page
- About page
- Fitness classes
- Nutrition information
- Contact page

These pages are connected through a navigation structure and share consistent layout and styling.
