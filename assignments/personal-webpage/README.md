# 👤 Personal Webpage

A personal introduction website created as part of **T INFO 230 – Web Development** coursework at the **University of Washington Tacoma**.

This assignment focuses on building a simple personal website while learning the fundamentals of HTML structure, page layout, and organizing web content.

## 📖 Project Overview

The Personal Webpage assignment introduces the process of creating a simple website that presents personal information and interests.

The website demonstrates how webpages can be structured to present information clearly using HTML elements, links, and images.

The project includes:

- a homepage introducing the author
- structured content sections
- image integration
- basic navigation
- organized webpage content

The goal is to practice **structuring and presenting information on the web using HTML**.

Webpages are built from HTML elements that define the structure and meaning of the content, such as headings, paragraphs, lists, and links. :contentReference[oaicite:0]{index=0}
