# 🎯 Hobby Website

A themed multi-page website created as part of **T INFO 230 – Web Development** coursework at the **University of Washington Tacoma**.

This project demonstrates foundational front-end web development skills including **HTML document structure, multi-page navigation, image integration, and basic layout techniques**.

---

View the site here:

https://jamdanie.github.io/uw-web-development/assignments/hobby-website/

---

## 📖 Project Overview

The Hobby Website presents a simple informational website built around a personal interest or hobby.  
The goal of the assignment was to practice organizing web content into multiple connected pages while maintaining a consistent structure and design.

The project includes:

- a homepage introducing the hobby
- multiple informational pages
- images and media integration
- consistent navigation between pages
- organized website assets

This assignment emphasizes **structure, organization, and understanding how websites are assembled from multiple files**.
