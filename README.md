T INFO 230 – Web Development Coursework

Course assignments and projects completed for T INFO 230: Web Development at the University of Washington Tacoma.

This repository contains introductory front-end web development exercises focusing on HTML structure, CSS styling, page layout, and multi-page website design.

These projects represent foundational work that supports later coursework in web applications, information systems, and software development.

Topics Covered

Projects in this repository demonstrate:

HTML document structure

Multi-page website navigation

External CSS styling

Page layout and formatting

Image integration and asset organization

Basic semantic HTML practices

Introductory web accessibility considerations

Static website organization and project structure

Purpose

The purpose of this repository is to document early coursework and demonstrate the progression of foundational web development skills within an academic setting.

These assignments form the groundwork for more advanced projects involving dynamic web applications, backend services, and modern web frameworks.
